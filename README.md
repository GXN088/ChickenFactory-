# ChickenFactory-
Напишем Фабрику (Factory) по производству кур (Hen):

Создай класс Hen.
Сделай его абстрактным.
Добавь в класс абстрактный метод int getCountOfEggsPerMonth().
Добавь в класс метод String getDescription(), который возвращает строку "Я - курица.".
Создай класс RussianHen, который наследуется от Hen.
Создай класс UkrainianHen, который наследуется от Hen.
Создай класс MoldovanHen, который наследуется от Hen.
Создай класс BelarusianHen, который наследуется от Hen.
В каждом из четырех последних классов напиши свою реализацию метода getCountOfEggsPerMonth.
Методы должны возвращать количество яиц в месяц от данного типа куриц.
В каждом из четырех последних классов напиши свою реализацию метода getDescription.
Методы должны возвращать строку вида:
<getDescription() родительского класса> + <" Моя страна - Sssss. Я несу N яиц в месяц.">
где Sssss - название страны
где N - количество яиц в месяц

В классе HenFactory реализуй метод getHen, который возвращает соответствующую стране породу кур.
Все созданные классы должны быть в отдельных файлах.



Псевдокод:

```
// ИНТЕРФЕЙС С КОНСТАНТАМИ
ИНТЕРФЕЙС Country
    КОНСТАНТЫ:
        UKRAINE = "Ukraine"
        RUSSIA = "Russia"
        MOLDOVA = "Moldova"
        BELARUS = "Belarus"


// АБСТРАКТНЫЙ БАЗОВЫЙ КЛАСС
АБСТРАКТНЫЙ КЛАСС Hen
    АБСТРАКТНЫЙ МЕТОД:
        getCountOfEggsPerMonth() : ЦЕЛОЕ ЧИСЛО
    
    МЕТОД:
        getDescription() : СТРОКА
            ВОЗВРАТ "Я - курица."


// КОНКРЕТНЫЕ РЕАЛИЗАЦИИ
КЛАСС RussianHen НАСЛЕДУЕТ Hen
    ПЕРЕОПРЕДЕЛИТЬ МЕТОД getCountOfEggsPerMonth():
        ВОЗВРАТ 5
    
    ПЕРЕОПРЕДЕЛИТЬ МЕТОД getDescription():
        БАЗОВЫЙ_ТЕКСТ = ВЫЗВАТЬ БАЗОВЫЙ МЕТОД getDescription()
        ЯЙЦА = ВЫЗВАТЬ getCountOfEggsPerMonth()
        ВОЗВРАТ БАЗОВЫЙ_ТЕКСТ + " Моя страна - Russia. Я несу " + ЯЙЦА + " яиц в месяц."


КЛАСС UkrainianHen НАСЛЕДУЕТ Hen
    ПЕРЕОПРЕДЕЛИТЬ МЕТОД getCountOfEggsPerMonth():
        ВОЗВРАТ 10
    
    ПЕРЕОПРЕДЕЛИТЬ МЕТОД getDescription():
        БАЗОВЫЙ_ТЕКСТ = ВЫЗВАТЬ БАЗОВЫЙ МЕТОД getDescription()
        ЯЙЦА = ВЫЗВАТЬ getCountOfEggsPerMonth()
        ВОЗВРАТ БАЗОВЫЙ_ТЕКСТ + " Моя страна - Ukraine. Я несу " + ЯЙЦА + " яиц в месяц."


КЛАСС MoldovanHen НАСЛЕДУЕТ Hen
    ПЕРЕОПРЕДЕЛИТЬ МЕТОД getCountOfEggsPerMonth():
        ВОЗВРАТ 9
    
    ПЕРЕОПРЕДЕЛИТЬ МЕТОД getDescription():
        БАЗОВЫЙ_ТЕКСТ = ВЫЗВАТЬ БАЗОВЫЙ МЕТОД getDescription()
        ЯЙЦА = ВЫЗВАТЬ getCountOfEggsPerMonth()
        ВОЗВРАТ БАЗОВЫЙ_ТЕКСТ + " Моя страна - Moldova. Я несу " + ЯЙЦА + " яиц в месяц."


КЛАСС BelarusianHen НАСЛЕДУЕТ Hen
    ПЕРЕОПРЕДЕЛИТЬ МЕТОД getCountOfEggsPerMonth():
        ВОЗВРАТ 11
    
    ПЕРЕОПРЕДЕЛИТЬ МЕТОД getDescription():
        БАЗОВЫЙ_ТЕКСТ = ВЫЗВАТЬ БАЗОВЫЙ МЕТОД getDescription()
        ЯЙЦА = ВЫЗВАТЬ getCountOfEggsPerMonth()
        ВОЗВРАТ БАЗОВЫЙ_ТЕКСТ + " Моя страна - Belarus. Я несу " + ЯЙЦА + " яиц в месяц."


// ФАБРИКА
КЛАСС HenFactory
    СТАТИЧЕСКИЙ МЕТОД getHen(страна: СТРОКА): Hen
        ЕСЛИ страна РАВНО Country.BELARUS ТОГДА
            ВОЗВРАТ НОВЫЙ BelarusianHen()
        ИНАЧЕ ЕСЛИ страна РАВНО Country.RUSSIA ТОГДА
            ВОЗВРАТ НОВЫЙ RussianHen()
        ИНАЧЕ ЕСЛИ страна РАВНО Country.UKRAINE ТОГДА
            ВОЗВРАТ НОВЫЙ UkrainianHen()
        ИНАЧЕ ЕСЛИ страна РАВНО Country.MOLDOVA ТОГДА
            ВОЗВРАТ НОВЫЙ MoldovanHen()
        ИНАЧЕ
            ВОЗВРАТ НИЧЕГО


// ОСНОВНАЯ ПРОГРАММА
КЛАСС Solution
    МЕТОД main():
        курица = HenFactory.getHen(Country.BELARUS)
        ВЫЗВАТЬ курица.getCountOfEggsPerMonth()
```

Краткое описание структуры:

1. Интерфейс Country - содержит константы названий стран
2. Абстрактный класс Hen - базовый класс для всех кур с абстрактным методом получения количества яиц
3. Конкретные классы кур (4 класса) - реализуют специфичное для каждой страны количество яиц и описание
4. Фабрика HenFactory - создает объекты кур в зависимости от страны
5. Решение (Solution) - демонстрирует использование фабрики

Примечание: В реальном Java коде используется паттерн "Фабричный метод", где логика создания объектов инкапсулирована в отдельный класс.
